# Pharm0

**Mechanism-grounded synthetic data for drug discovery.**

Pharm0 is an AI-native synthetic-data factory for pre-clinical pharmaceutical research. We build the data, auditably, with agents.

## Repos

| Repo | Purpose |
|---|---|
| [`pharm0`](https://github.com/pharm0-ai/pharm0) | Synthetic-data platform — the product. Agent runtime, SDK, eval harness, data APIs. |
| [`pharm0-web`](https://github.com/pharm0-ai/pharm0-web) | Public website + waitlist + lab subsite shell. |
| [`pharm0-lab`](https://github.com/pharm0-ai/pharm0-lab) | Drug-discovery workflows. Consumer of the platform via API. |
| [`pharm0-contracts`](https://github.com/pharm0-ai/pharm0-contracts) | Shared API + event schemas (Apache-2.0). |
